# 🏥 UPA Sistema

Sistema desenvolvido em linguagem C para simular o atendimento de uma Unidade de Pronto Atendimento (UPA), utilizando estruturas de dados como fila de prioridade e pilha.

---

## 📌 Funcionalidades

- Cadastro de pacientes
- Fila de prioridade
- Atendimento por gravidade
- Controle de ordem de chegada
- Desfazer último atendimento
- Organização dinâmica da fila

---

## 🧠 Estruturas Utilizadas

- Lista Encadeada
- Fila de Prioridade
- Pilha

## ▶️ Como Executar

### Clonar o projeto

```bash
git clone https://github.com/Rmonn25/upa-sistema.git
```

### Entrar na pasta

```bash
cd upa-sistema
```

### Compilar

```bash
gcc src/*.c -o programa
```
### Configuração no cmd

```bash
chcp 65001
```

### Executar

```bash
programa.exe
```
